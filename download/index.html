Gpg and checksum verification

How can I verify integrity (checksum and signature) of my downloaded Canoe installation package? You will need the gpg and md5sum software.

On Linux

Gpg can be installed using the package manager of your distribution. Add the Public key of Canoe to your keyring:

gpg --keyserver hkp://pgp.mit.edu --recv-keys E7ADC266
  • Download the Canoe zip and the relative zip.sig to the same folder.
  • Change in the download folder (or whereever you saved them - delete downloads of older version first!) and run in terminal window:
cd ~/Downloads
gpg --verify canoe-linux64*.zip.sig canoe-linux64*.zip
md5sum canoe-linux64*.zip

On Mac OSx

  • Install gpgtools; Documentation can be found here.
  • Display the Canoe public key in your browser.
  • Press cmd + A to select all, then copy / paste that information into your GPG Keychain main window - that will import our public key.
  • Run the following commands in Terminal.app:
gpg --verify canoe-linux64*.zip.sig canoe-linux64*.zip
md5sum canoe-linux64*.zip
